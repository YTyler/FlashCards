JavaScript Flashcards!

JavaScript (JS)

What does this mean?

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

What does this mean?

Operators in JavaScript are very similar to operators that appear in other programming languages. The definition of an operator is a symbol that is used to perform an operation. Most often these operations are arithmetic (addition, subtraction, etc), but not always.

Variables

What does this mean?

JavaScript Variable. Variable means anything that can vary. JavaScript includes variables which hold the data value and it can be changed anytime. JavaScript uses reserved keyword var to declare a variable. ... You can assign a value to a variable using equal to (=) operator when you declare it or before using it.

Functions

What does this mean?

Functions are one of the fundamental building blocks in JavaScript. A function is a JavaScript procedure—a set of statements that performs a task or calculates a value. To use a function, you must define it somewhere in the scope from which you wish to call it.

Methods

What does this mean?

JavaScript methods are actions that can be performed on objects. A JavaScript method is a property containing a function definition.

Arguments

What does this mean?

Arguments is an Array -like object accessible inside functions that contains the values of the arguments passed to that function. ... Note: “Array-like” means that arguments has a length property and properties indexed from zero, but it doesn't have Array 's built-in methods like forEach() and map().

Parameters

What does this mean?

Javascript is a functional language meaning that functions are the primary modular units of execution. Functions are obviously very important in Javascript. ... Parameters are variables listed as a part of the function definition. Arguments are values passed to the function when it is invoked.

Return

What does this mean?

The return statement stops the execution of a function and returns a value from that function.
